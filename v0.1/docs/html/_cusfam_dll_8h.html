◆ CUSFAM_API

#define CUSFAM_API

CUSFAM DLL Interface Header.

This header file provides the complete C++ DLL interface for the CUSFAM nuclear reactor physics simulation code. It includes classes for steady-state calculations, transient analysis, xenon dynamics, shutdown margin analysis, and various reactor operations.

The DLL interface wraps the original CUSFAM code to provide a clean, platform-independent API that can be used from C++, C, and other programming languages.
